# 🤖 AutoMate — Microsoft 365 Copilot Agent

> Work Smarter. Not Harder.

## 🏆 Microsoft Agents League Hackathon 2026

AutoMate is an enterprise AI agent built on Microsoft 
Copilot Studio that automates repetitive daily workplace 
tasks — saving employees 2+ hours every single day.

## ✨ Features

- 📝 Auto-summarizes Teams meetings
- ✅ Creates tasks in Microsoft Planner automatically  
- 🌅 Sends daily morning briefing digest
- 💬 Answers workplace questions from SharePoint
- 📊 Tracks productivity and time saved

## 🛠️ Built With

- Microsoft Copilot Studio
- Azure AI Foundry
- Microsoft Teams
- Microsoft Planner
- SharePoint
- Power Automate
- Microsoft Graph API
